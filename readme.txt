== Description ==

<h3>PagosWeb Payment Gateway for WooCommerce</h3> makes your website ready to use PagosWeb payment gateway to accept credit/debit cards on your ecommerce store in safe way.

== Installation ==

Easy steps to install the plugin:

1. Upload `pagosweb-payment-gateway-woocommerce` folder/directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce => Settings
4. On the "Settings" page, select "Payment Gateways" tab.
5. Under "Payment Gateways" you will find all the available gateways, select "PagosWeb" option
6. On this page you wil find option to configure the plugin for use with WooCommerce
7. Enter the API details provided by PagosWeb
